Text Analysis - Zipf's Law

The following Plots show my 'Properties of Text' final-year university project. In this Project Zipf's Law (a frequency-rank analysis of words in text) was done on number of corpora (bodies of words) taken from the indigenous languages of Yoruba, Igbo and Hausa.
I prepared the word datasets by scraping these indigenous words off of online websites in these langauges, such as BBC News (Yoruba, Igbo & Hausa), and YouVersion. The datasets were further organised and cleaned using python and natural language processing tools within NLTK library. Text bodies of more than 1 million unique words in each language were created and in each language they were analysed and seen to follow Zipf's Law.
Each button leads to a different part of the GitHub directory for this project
